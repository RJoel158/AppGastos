# app_compras

Aplicación de lista de compras desarrollada con Flutter y SQLite.

## Características

- ✅ Agregar productos con nombre, precio y cantidad
- ✅ Incrementar/decrementar cantidad de productos
- ✅ Cálculo automático del total de compra
- ✅ Base de datos SQLite para persistencia
- ✅ Interfaz moderna con Material Design 3
- ✅ Eliminación individual y masiva de productos

## Instalación

1. Asegúrate de tener Flutter instalado
2. Clona o descarga este proyecto
3. Ejecuta en la terminal:

```bash
flutter pub get
flutter run
```

## Tecnologías

- Flutter 3.0+
- SQLite (sqflite)
- Material Design 3
